Maester Toolkit provides you necessary widgets for better and effective LMS theme.

== Description ==

Maester Toolkit is a lightweight and safe plugin that generates 2 much-necessary custom elementor addons and tutor lms shortcodes. This plugin is optimized for Maester-Lite’s themes but also works great with other themes.

= Widgets & Shortcode included in  Maester Toolkit =

* Maester Courses -- This widget lets you show tutor lms courses.
* Maester Category -- This widget lets you show tutor lms categories.
* [maester_course] -- This shortcode lets you show tutor lms courses.



== Installation ==
**From your WordPress dashboard**

1. Visit 'Plugins > Add New',
2. Search for 'Maester Toolkit' and install the plugin.
3. Activate ‘Maester Toolkit’ from your Plugins page.
